## Comandos usados

- `ls` lista archivos
- `ls -l` lista archivos con mas info
- `ls -la` muestra todo, con los ocultos
- `echo` envia texto, inclusive a un archivo, 
- `cat` muestra el contenido del archivo
- `man` manual, ejemplo `man cat`
